# Python general knowledge quiz
How to run the quiz:
1) Clone the repository and open the project in your code editor
2) Once the code is open, click run and it should appear in the terminal/console where you can enter your answers to the question
3) At the end the terminal will indicate when you've completed the quiz and your final score

## How the code works
This Python quiz uses modular code to store questions in a dictionary and loops through the data structure to access questions and answers to check if they're correct. The code will count how many questions are correct and give a final score at the end
